# Computer Architecture and Hardware Learning Path

This repository contains structured roadmaps and key insights into how computers work at a low level, with a focus on computer architecture and hardware.

## Available Roadmaps

Currently, the `roadmap` directory provides comprehensive guides for learning hardware fundamentals:

- **[Computer Hardware Learning Roadmap](./roadmap/computer_hardware_learning_roadmap.md)**: A structured layer-by-layer guide to learning computer fundamentals, CPU/GPU architecture, performance concepts (TDP, IPC), and comparisons between Intel, AMD, and Apple Silicon. Includes curated resources (books, videos, articles) and practical exercises.
- **[Hardware Learning Roadmap](./roadmap/hardware_learning_roadmap.md)**: A detailed, phase-based path from foundational mental models of CPUs to applied understanding of spec sheets, architecture deep dives, GPU fundamentals, and emerging topics like chiplets, NPUs, and RISC-V.

## Core Concepts

The objective of this project is to build a solid mental model of:
- The instruction cycle (fetch-decode-execute)
- CPU architectures (x86 vs ARM, RISC vs CISC)
- Multicore processing and cache hierarchies
- Hardware performance metrics (IPC, TDP, clock speeds)
- Modern computing architectures (Apple Silicon efficiency, modular chiplets, NPUs)

Explore the roadmaps to begin your path from zero to deeply understanding computer hardware!
